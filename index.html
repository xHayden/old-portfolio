Karten

Karten is a tool built in React to quickly create
flashcards for foreign languages.

Karten supports any languge supplied by the Yandex API
and is able to convert to Quizlet Study Sets.

Twitch Checker

The Asynchronous Twitch Username checker is capable of checking
thousands of Twitch usernames in record times, giving
you the ability to get the best usernames possible.

Twitch does not publicly show when usernames will become available,
so such as system is necessary to get the username you want.

The project is currently closed source, however the project will
eventually be transformed into a fully featured website.
